1. Form Tag: The <form> tag is used to create an HTML form.
    It has attributes like action, method, and enctype.

2. Input Types: The <input> element is the most used form element. It can be displayed in many ways, depending on the type attribute.
    Some common types include text, password, submit, radio, checkbox, button, file, hidden, image, reset, date, etc.

3. Text Fields: <input type="text"> defines a one-line input field that a user can enter text into.
    It's often used when a single line of text is expected.

4. Radio Buttons: <input type="radio"> defines a circular selection button in an HTML form.
    You can select one, and only one, option in a radio button group.

5. Checkboxes: <input type="checkbox"> defines a checkbox, which the user can toggle on or off.

6. Submit Button: <input type="submit"> defines a button for submitting the form data to a form-handler.

7. Action Attribute: The action attribute defines the action to be performed when the form is submitted. 
    Usually, the form data is sent to a page on the server when the user clicks on the submit button.

8. Method Attribute: The method attribute defines the HTTP method to be used when sending form-data. 
    The method can be get or post.

9. Select List: The <select> element is used to create a drop-down list. 
    The <option> tags inside the <select> element define the available options in the list.

10. Textarea: The <textarea> tag defines a multi-line text input control.
    It can be used when you want to allow users to type a sizeable amount of text.
